## 🔧 技术实现

### Hooks
- `useMobile()`: 检测设备类型和屏幕尺寸
- `useTouch()`: 触摸手势支持
- `useCamera()`: 相机权限检测（用于 Cimbar）
- `useBiometric()`: 生物识别支持

### CSS 类
- `.mobile-only`: 仅在移动端显示
- `.desktop-only`: 仅在桌面端显示
- `.safe-area-top`: 顶部安全区域
- `.safe-area-bottom`: 底部安全区域
- `.modal-container`: 移动端全屏模态框

## 📊 性能优化

1. **条件渲染**: 根据设备类型渲染不同组件
2. **懒加载**: 模态框按需加载
3. **CSS 优化**: 使用 Tailwind 响应式类
4. **触摸优化**: 减少不必要的重绘
